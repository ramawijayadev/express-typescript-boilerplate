Backend Starterkit

Dokumentasi induk, standar teknis, dan filosofi pengembangan aplikasi backend production-grade.

Kitab ini adalah titik awal (Entry Point). Starterkit ini dirancang untuk mencapai keseimbangan antara Kecepatan Development (MVP) dan Kestabilan Jangka Panjang (Scalability).

üöÄ Quick Start Philosophy

Bagaimana cara menggunakan starterkit ini secara efektif?

  1. Baca Architecture: Pahami dulu "Kenapa" kita menyusun folder seperti ini.
  2. Install Dependency: Ikuti panduan instalasi pnpm di Dependency Handbook.
  3. Ikuti Convention: Saat menulis fitur baru, contek struktur nama file dan kode dari Convention Handbook.
  4. Gunakan Core Feature: Jangan buat Auth atau Upload dari nol, gunakan modul yang didefinisikan di Feature Handbook.
